# Example Templates

This directory contains ready-to-use templates for online code editors.

## How to Use

### CodeSandbox

1. Go to [codesandbox.io](https://codesandbox.io)
2. Click "Create Sandbox" → "React"
3. Copy the contents of `codesandbox-template.js` into `App.js`
4. Install the package: `npm install saas-effects` (or use the code directly)
5. Start coding!

### StackBlitz

1. Go to [stackblitz.com](https://stackblitz.com)
2. Click "New Project" → "React"
3. Copy the contents of `stackblitz-template.js` into `App.jsx`
4. Install the package: `npm install saas-effects` (or use the code directly)
5. Start coding!

## Templates Included

- **codesandbox-template.js** - Complete React example with both components
- **stackblitz-template.js** - Complete React example optimized for StackBlitz

Both templates showcase:
- BackgroundGradient with various props
- CardWithCornerAccent with different corners
- Preset usage
- Custom colors
- Multi-color gradients
- Dark mode
- Custom angles

## Local Development

If you want to test these examples locally:

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```
